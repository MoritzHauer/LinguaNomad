# LinguaNomad Vision

## Problem

Most language-learning products are optimized for large markets, shallow early engagement, or vocabulary drilling.
That leaves learners of underrepresented languages with fragmented materials, weak progression, and little support for script, morphology, or authentic communicative use.

## Product Direction

LinguaNomad is an open-source language-learning app that starts with Kyrgyz and is designed for serious learners who want durable progress rather than light entertainment.
The app combines structured lessons, spaced retrieval, and meaningful tasks, while keeping the learning experience usable on a phone and resilient offline.

## Target Learner

The first target user is a serious long-term Kyrgyz learner, including heritage learners and motivated self-directed learners.
The product should still be beginner-accessible, but it should not flatten the language into toy exercises.

## Core Value Proposition

LinguaNomad should help a learner move from zero or rusty knowledge toward real Kyrgyz comprehension and use through:
- Clear script and pronunciation support
- Lesson sequences built around useful meaning, not isolated word lists
- Review loops that retain what matters
- Tasks that require understanding and producing language in context
- Grammar help where it removes genuine friction

## Product Pillars

### Comprehensible Input

Learners should receive graded input that is understandable but not trivial.
This includes short dialogues, phrases, micro-readings, and audio-backed examples.

### Spaced Retrieval

Important items should recur on a schedule shaped by learner performance.
Review is a support system for retention, not the entire product.

### Task-Based Practice

Learners should regularly complete tasks with a clear communicative outcome, such as introducing themselves, handling a greeting exchange, or interpreting simple situational language.

### Explicit Support

Kyrgyz-specific difficulty points such as Cyrillic familiarity, vowel harmony, cases, politeness, and sentence structure should be explained when needed.

## Differentiators

LinguaNomad should differ from mass-market apps in five ways:
- It is built for underrepresented languages from the start.
- It treats offline use as a real requirement.
- It models language content beyond simple word pairs.
- It respects licensing boundaries for public content.
- It supports serious progression rather than engagement theater.

## Non-Goals For The MVP

The MVP should not attempt to:
- Support many languages at equal depth
- Offer social feeds, leagues, or streak-first design
- Promise speech scoring or AI tutoring before the core loop works
- Replace careful editorial content with weak automated generation

## Success Criteria

The MVP is successful if a motivated learner can:
- Complete the first content pack on mobile
- Study and review offline
- Understand the script and core pronunciation conventions
- Retain foundational vocabulary and sentence patterns
- Complete simple communicative tasks without leaving the app
