# LinguaNomad Content Sources

## Purpose

This document tracks likely Kyrgyz content sources, their role in the product, and the operational caveats that affect ingestion.

## Source Categories

### Open Or Likely Reusable References

| Source | Likely Use | Caveat |
| --- | --- | --- |
| Tatoeba | Example sentences, translations, some audio links | Quality and coverage are uneven; attribution and per-entry license handling must be checked carefully |
| Wiktionary | Lexical references, transliteration rules, usage clues | Entry completeness varies; public ingestion should capture source attribution |
| Wikimedia and Wikivoyage phrase resources | Phrasebook-style starter material and cross-checking | Not sufficient alone for a full course |
| Omniglot references | Script and pronunciation orientation | Better for reference than course content |

### Restricted Editorial References

| Source | Likely Use | Caveat |
| --- | --- | --- |
| Peace Corps Kyrgyz Language Manual (1997) | Coverage planning, dialogue mining, topic sequencing, grammar cross-checking | Treat as non-public editorial input until redistribution rights are confirmed; ship original LinguaNomad wording, notes, and tasks rather than copied lesson text |

### Original Authored Content

Original content is likely required to create a coherent starter course.
This includes:
- Unit narratives
- Grammar notes
- Exercise prompts
- Task flows
- Curated transliteration guidance

### Prototype-Only Content

Mixed-source internal prototyping is allowed for experimentation, but unclear-license material must stay outside the public repository and any shipping content packs.

## Source Evaluation Rules

Before a source enters the public repository, record:
- Source URL
- License or usage terms
- Attribution requirements
- Commitable status
- Ingestion notes
- Quality notes

## Content Priorities For Kyrgyz MVP

The first public content pack should prioritize:
- Script familiarity
- Greetings and introductions
- Politeness and address forms
- Basic identity statements
- High-utility sentence patterns
- Beginner-safe explanations of vowel harmony and case behavior

## Operational Note

The product should not rely on source scraping as the main content strategy.
Open sources are inputs into a curated learning product, not a substitute for editorial design.

For the Peace Corps Kyrgyz manual specifically, the safest operational model is: use it to identify beginner communication goals, vocabulary domains, and grammar sequencing, then convert those into original LinguaNomad units backed by explicit source metadata.
