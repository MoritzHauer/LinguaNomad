# Kyrgyz Resources

## Knowledge

- [Peace Corps Kyrgyz courses via Live Lingua](https://www.livelingua.com/project/peace-corps/Kyrgyz)
  Best current structured beginner source found in this pass. Use for: staged beginner lessons, dialogs, and follow-on self-study after each lesson here.
- [Omniglot: Kyrgyz language and alphabet](https://www.omniglot.com/writing/kyrgyz.htm)
  Compact alphabet and pronunciation overview with a sample text. Use for: script inventory, rough pronunciation cues, and quick cross-checks.
- [Wikipedia: Kyrgyz alphabets](https://en.wikipedia.org/wiki/Kyrgyz_alphabets)
  Broad overview of script history and the full correspondence chart. Use for: checking the 36-letter inventory and script context; verify against stronger sources when precision matters.

## Wisdom (Communities)

- Family members who already speak Kyrgyz
  Highest-value real-world feedback loop for this mission. Use for: pronunciation reality checks, useful household vocabulary, and noticing what language is actually needed.
- Local Kyrgyz community events or meetups, if available
  Use for: live listening, polite greetings, and testing whether reading practice transfers to real people.

## Gaps

- Need a stable high-trust source with native audio for the letter names and minimal pairs.
- Need a reliable beginner-friendly Kyrgyz dictionary recommendation once lessons move beyond script work.
