Lesson 0048

Definite Future / Present Tense

You will be able to form the tense that covers both habitual present and certain future, and see how it completes the Kyrgyz tense system.

Mission: communication Skill: grammar Time: 25–30 minutes

Builds on: Lesson 0047 — Recent Past Tense (Eyewitness)

The Present-Future Tense

This tense is the workhorse of Kyrgyz. It covers habitual actions in the present and actions that will definitely happen in the future. The pattern is:

verb stem + -а/-е/-й + personal ending

The connecting vowel is determined by two rules:

  • Stems ending in a consonant: add -а (back vowel stems) or -е (front vowel stems)
  • Stems ending in a vowel: add -й

Personal Endings

PersonEndingExample (бар- to go)English
мен-мБарамI go / I will go
сен-сың/-сиңБарасыңYou go / will go (informal)
сиз-сыз/-сизБарасызYou go / will go (formal)
ал-тБаратHe/she goes / will go
биз-быз/-бизБарабызWe go / will go
силер-сыңарБарасыңарYou all go / will go
алар-шат/-шетБарашатThey go / will go

Key distinction: 3rd person singular takes -т, not -м. Do not confuse Барат (he/she goes) with Барам (I go).

What This Tense Covers

Unlike English, Kyrgyz uses a single form for three related meanings:

MeaningExampleEnglish
Habitual presentМен күн сайын базарга барам.I go to the bazaar every day.
Definite futureМен эртең барам.I will go tomorrow.
Near certaintyАл кела жатат.He is coming / will come.

The Full Tense System

You now have all three core tenses. Each encodes a distinct piece of information:

TensePatternExampleEnglish
Eyewitness paststem + -ды + endingБардымI went (I was there)
Present / Futurestem + -а/-е + endingБарамI go / I will go
Hearsay paststem + -ыптырБарыптырApparently he went

One form, two time zones

English speakers often look for a separate "will" to mark the future. In Kyrgyz, the present-future tense does both jobs. Context — a time word like эртең (tomorrow) or күн сайын (every day) — tells the listener which meaning is intended. This is common across Turkic languages.
